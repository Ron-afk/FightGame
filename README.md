# FightGame
 
Mini web based fight game

Control:

    Player 1: 
    
        left: a
        
        right: d
        
        jump: w
        
        attack: space
        
    Player 2:
    
        left: left arrow
        
        right: right arrow
        
        jump: up arrow
        
        attack: down arrow

Deployed with GitHub pages

https://ron-afk.github.io/FightGame/
